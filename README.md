# Spam Mail Detector

This project builds a machine learning model that classifies emails as **Spam** or **Ham (Not Spam)** using Natural Language Processing.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression

## Dataset
SMS Spam Collection Dataset

## Workflow
1. Load dataset
2. Data preprocessing
3. Convert labels to numeric
4. Text vectorization using TF-IDF
5. Train Logistic Regression model
6. Evaluate model performance
7. Predict new messages

## Example Prediction

Input:
"You won the draw, claim your prize"

Output:
Spam

## Accuracy
~96% accuracy on test data.
